
# Open Ottawa

Welcome to the Ottawa Building Explorer project! This iOS application allows users to explore a curated list of buildings in Ottawa that are open to the public.


## Features

- Building list
- Building details
- Interactive Map
- Favorites
- Recommend a New Building


## Tech Stack

- Swift
- SwiftUI
- MapKit
- Network Monitor
- Location Manager
- Firebase
- Safari Services
## Requirements

- Xcode 12.0 or later
- Swift 5.0
- iOS 16.0 or later
